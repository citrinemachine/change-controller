# change-controller
Quickly change yuzu controllers

##Install and Run
1. Install current version of Python 3 
2. Download yuzu_controller_quick_change.py
3. Open a command propmt or Powershell terminal where you downloaded yuzu_controller_quick_change.py
4. Type `python yuzu_controller_quick_change.py`
5. Select options from the menu.

##Usage
1. Choose "Backup Config" option to back up your config file
2. Setup one of your controller configs in Yuzu (tested in custom)
3. Then save the controller config with "Save Controller Configs" option
4. Setup the next controller configs
5. Then save the controller config with "Save Controller Configs" option again
6. When you want to change between controller configs choose the "Load Configs Option"

##Optional Usage
- See a list of saved controllers with "See list of Saved Configs" option
- See raw controller config data with "See Config Data" option


##Notes
- This program doesn't work without YUZU installed.
